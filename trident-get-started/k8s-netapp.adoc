= Kubernetes 與 NetApp 產品的整合
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp 儲存產品組合與 Kubernetes 叢集的許多不同面向整合，提供進階資料管理功能，從而增強 Kubernetes 部署的功能、能力、效能和可用性。



== Trident

https://docs.netapp.com/us-en/trident/["Trident"^]讓企業能夠更輕鬆地管理、保護及移動在 Kubernetes 上執行的資料密集型容器化工作負載，無論是在公有雲內部、跨公有雲之間或本地環境中。Trident 利用經過驗證且功能強大的 NetApp 儲存產品組合，在公有雲和本地環境中提供及配置持久容器儲存。此外，它還提供一系列豐富的、以應用程式為導向的進階資料管理功能，例如快照、備份和還原、活動日誌以及主動複製，以滿足 Kubernetes 工作負載的資料保護、災難 / 資料復原、資料稽核和遷移等各種使用案例。



== ONTAP

ONTAP 是 NetApp 多協定統一儲存作業系統，可為任何應用程式提供進階資料管理功能。ONTAP 系統具有全快閃、混合或全 HDD 配置，並提供多種不同的部署模型，包括工程硬體（FAS 和 AFF）、白盒（ONTAP Select）和純雲端（Cloud Volumes ONTAP）。


NOTE: Trident 支援上述所有 ONTAP 部署模型。



== Cloud Volumes ONTAP

http://cloud.netapp.com/ontap-cloud?utm_source=GitHub&utm_campaign=Trident["Cloud Volumes ONTAP"^] 是一款僅限軟體的儲存設備，可在雲端中執行 ONTAP 資料管理軟體。您可以使用 Cloud Volumes ONTAP 處理生產工作負載、災難復原、DevOps、檔案共用和資料庫管理。它透過提供儲存效率、高可用性、資料複製、資料分層和應用程式一致性，將企業儲存擴展到雲端。



== Amazon FSx for NetApp ONTAP

https://docs.aws.amazon.com/fsx/latest/ONTAPGuide/what-is-fsx-ontap.html["Amazon FSx for NetApp ONTAP"^] 是一項完全託管的 AWS 服務，可讓您啟動並執行由 NetApp ONTAP 儲存作業系統驅動的檔案系統。FSx for ONTAP 讓您能夠利用您熟悉的 NetApp 特性、效能和管理功能，同時享受在 AWS 上儲存資料的簡易性、敏捷性、安全性和可擴充性。FSx for ONTAP 支援 ONTAP 檔案系統的許多特性和管理 API。



== Element 軟體

https://www.netapp.com/data-management/element-software/["元素"^] 使儲存管理員能夠整合工作負載，從而確保效能並簡化和最佳化儲存佈局。結合用於自動化儲存管理各方面的 API，Element 讓儲存管理員能夠事半功倍。



== NetApp HCI

https://www.netapp.com/virtual-desktop-infrastructure/netapp-hci/["NetApp HCI"^] 透過自動化日常任務，簡化資料中心的管理和擴展，使基礎設施管理員能夠專注於更重要的功能。

NetApp HCI 完全由 Trident 支援。Trident 可以直接針對底層 NetApp HCI 儲存平台，為容器化應用程式配置和管理儲存裝置。



== Azure NetApp Files

https://azure.microsoft.com/en-us/services/netapp/["Azure NetApp Files"^] 是由 NetApp 提供支援的企業級 Azure 檔案共用服務。您可以在 Azure 中原生運作要求最嚴苛的基於檔案的工作負載，並獲得您期望從 NetApp 獲得的卓越效能和豐富的資料管理功能。



== Google Cloud NetApp Volumes

https://cloud.netapp.com/cloud-volumes-service-for-gcp?utm_source=GitHub&utm_campaign=Trident["Google Cloud NetApp Volumes"^] 是一項雲端原生檔案服務，可透過 NFS 提供全快閃效能的 NAS 磁碟區。此服務可讓包括舊版應用程式在內的任何工作負載在 GCP 雲端中執行。它提供完全託管的服務，可提供一致的高效能、即時複製、資料保護，以及對 Google Compute Engine （GCE） 執行個體的安全存取。
